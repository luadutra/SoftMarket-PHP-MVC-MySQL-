# Sitema de Vendas (SoftMarket) #


Desenvolvimento:

- PHP 7.0.33
- MySQL
- MVC

Objetivo:

Este sistema objetiva a simulação de vendas de um supermercado.
O sistema apresenta o valor de cada produto multiplicado pela quantidade adquirida e a quantidade paga de imposto em cada um dos itens, 
gerando um montante do valor da compra e um montante do valor dos impostos.

Funcionalidades:

- Cadastro de produtos (inserção, edição e exclusão).
- Cadastro dos tipos de cada produto (inserção, edição e exclusão).
- Vendas (inserção de produto, remoção de produto, cancelamento da venda e finalização da venda).

Configurações:

- Altere os dados da conexão com o banco de dados no arquivo 'DB_connect.php' conforme os seus dados locais.
